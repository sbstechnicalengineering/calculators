About This Tool

The SBS Finish Guide is an internal engineering reference for selecting, comparing, and specifying fastener surface finishes. Data is sourced from a Google Sheets flat file — no code changes are needed to add or update finishes.

To request a new finish, correction, or addition, contact your SBS administrator and edit the Google Sheet directly.

Data Source Configuration

This tool reads from two Google Sheets tabs published as CSV. To configure:

  • Open the Google Sheet → File → Share → Publish to web
  • Publish each tab as CSV format and copy the URL
  • Paste into SHEET_FINISHES_URL and SHEET_CODES_URL at top of script

Google Sheet Column Reference

Sheet 1 — Finishes (one row per finish):

id | category | name | subtitle | description | he | rohs_note | cost | standard | callout | note | color | img_url | materials

Sheet 2 — Codes (one row per spec code):

finish_id | code | thk | conversion | white | red | rohs

color values: silver | yellow | black | olive | chrome | gray  ·  materials: comma-separated list e.g. steel,stainless  ·  img_url: full GitHub raw URL to finish photo
